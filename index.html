I am a co-creator and software engineer of CushionDB - an open-source data management framework for building offline-first Progressive Web Apps.

CushionDB is an open-source data management framework for building offline-first Progressive Web Apps.

CushionDB was built using Node.js and integrates a variety of offline-first and Progressive Web App tools. CushionDB is made up of two main components - an NPM package for the client-side and two docker images for the backend which are deployed using docker-compose.
